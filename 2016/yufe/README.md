##	ECMAScript
	标准演进

- ES3: 1999年12月
- ~~ES4~~: 废弃
- ES5: 2009年12月

- ES2015 (ES6) 去年6月
- ES2016 (ES7) 今年6月
- ES2017 (ES8) 明年6月

## ES6 主要特性总结

  - module
  - class
  - const/let
  - arrow function
  - Symbol, well-known symbols
  - iterator, generator, for-of
  - Promise

  - destructuring, default value
  - spread, rest param
  - template string
  - Set/Map
  - Proxy
  - 内建对象的扩展
  - ……
